# Al-Powered Email Tone Checker Chatbot

## Description

This is a simple Al chatbot that checks the tone of emails.

It can detect if an email is "Friendly, Formal, Apologetic, or Aggressive and gives suggestions to improve the email.

The project uses "Streamlit" for the interface and "Hugging Face Transformers" for tone detection.

## Steps to Run

1. "Install required libraries"

Make sure you have Python installed, then run:

pip install -r requirements.txt

2. "Run the app"

streamlit run app.py

"Paste your email"
